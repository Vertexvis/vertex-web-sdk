Interface AdaptiveRenderingSettings

Settings to configure the adaptive rendering of images. If enabled, the performance of delivering images is analyzed, and will be adjusted to deliver frames quicker to the client.

Optional windowSize

windowSize: undefined | number
  • Defined in settings.ts:103

The window size to use for smoothing based adaptive rendering methods. This applies to: smoothed-median, median-of-smoothed-median, average-of-smoothed-median. Higher window sizes will result in values that are more smoothed, and have less "wave" like behavior to image quality. However, higher window sizes will take longer to respond to changes in network conditions.
