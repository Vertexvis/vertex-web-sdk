    Interface Matrix4AsObject

    An object representation of a Matrix4, where each value is represented as:

    m11 m12 m13 m14
    m21 m22 m23 m24
    m31 m32 m33 m34
    m41 m42 m43 m44

    Matrix4 arrays can be converted to an object using toObject.
